RETAS STUDIOとは

アニメーション制作には、RETAS STUDIOの後継ソフト「CLIP STUDIO PAINT（クリップスタジオ）」もご利用いただけます。 CLIP STUDIO PAINTではRETASシリーズを開発した際の技術を受け継ぎながら、新しい制作環境に合わせた新しい提案をいたします。　詳しくはこちら

はじめる　-Start-

一人でも多くの方に、アニメーションを制作していただくため、使いやすく、わかりやすいソフトウェアを目指しました。

簡単インストールで制作開始

RETAS STUDIOは4つのツールで構成されていますが、インストール作業は簡単。アプリケーションディスクをPCのDVDドライブに挿入するだけ。あとは画面に従ってインストール完了です。

公式Webサイトで腕をみがこう

「RETAS STUDIO」の公式サイトでは、各種情報や、アニメーション制作をサポートする様々な企画を随時ご用意していきます。

サンプルアニメ付きガイドブック「RETAS STUDIOでアニメを作ろう！」

パッケージにはガイドブック「RETAS STUDIOでアニメを作ろう！」を収録し、アニメーション制作をサポート。 実際にオリジナルサンプルアニメ『レタス戦隊スタジオン』の制作を行う工程を通して、アニメ制作を基礎からご紹介しています。
追加のサンプルデータを、「創作活動応援サイト CLIP」からダウンロードできます。

  • 創作活動応援サイト CLIP


つくる　-Create-

数多くのプロの現場で使われているツールを使って、オリジナルのアニメを作ろう！

プロの現場と同じワークフローを4つのツールで実現

「RETAS STUDIO」は、工程ごとに設計された4つのツールでプロのアニメーション制作現場を再現しています。そのため1人でも、RETAS STUDIOユーザー同士で分担して1つの作品を制作することもできます。すでにある素材やアナログで制作された素材を取り込むなど、使い方も自由自在。作画はやっぱり紙に描きたい、など自分にあった制作スタイルで作品を完成できます。

「CLIP 保管・共有する」を通じて遠く離れたユーザー同士での共同作業が簡単にできます。
詳細は、創作活動応援サイト 「CLIP」をご覧ください。

アニメーション制作のすべての工程をカバー

4つのツールに加え、Windows版には映像編集ソフト「Movie Edit Pro」（発売元：株式会社 ＡＨＳ）をパッケージに収録。
撮影した作品を編集し、ナレーションやBGMの入った本格的な作品を完成できます。
「Movie Edit Pro」は、DV/HDVテープへの書き出しはもちろん、iPod/PSPなどモバイル機器用ファイルの作成や、YouTubeへのアップロード機能も搭載しており、作った映像をすぐにインターネットへアップロードすることも可能です。

※ Mac OS X 版には同梱されていません。PCに標準インストールされている映像編集ソフト「iMovie」などのご使用をお勧めします。


ひろがる　-Extend-

Web用アニメから劇場用アニメまで幅広い作品を制作可能

もともとプロの世界で使われてきたツールのため、「RETAS STUDIO」は数秒のWeb用アニメから、劇場用公開アニメまで、幅広い作品を制作できます。

プロのアニメ制作を体験

「機動戦士ガンダム」シリーズを手がけた関田 修氏、「うる星やつら」、「機動警察パトレイバー」を手がけた高田 明美氏などの手による、オリジナル短編アニメ「魔法少女マイ」とそのメイキングデータを第1話から第3話まで収録。アニメ界のトップクリエイターが制作したデータで、プロのアニメーション制作を体験できます。
